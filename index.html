Research Experience

In this project, I built and released one thousand graph-based traffic accident benchmark datasets (TAB-1k). We also proposed a novel GNN framework to capture angular and directional information from road networks. I then comprehensively evaluate the proposed framework against fourteen state-of-the-art machine learning approaches using the created datasets. We demonstrate that our proposed model consistently outperforms the baselines.

This project involved simulating the process of data collection using taxis (e.g., obtaining the filling level of garbage bins’ capacity and humidity of vegetation along the roads). Based on an analysis of the real-world trajectories of taxis, We proposed three schemes to improve the deployment of data centers to increase the efficiency of data collection. This research proved the feasibility of utilizing taxis to collect useful data that, if applied in practice, has the potential to facilitate the maintenance of infrastructure.

This work investigates Information-Centric Networking (ICN), which is a future network architecture that has high efficiency, availability, and scalability. We devised two algorithms to improve the routing and placement of data packets. This work could improve network performance for some ICN applications. The experiments and simulations show that the proposed algorithms outperformed five existing algorithms in terms of delay, cache hits, and network traffic.
